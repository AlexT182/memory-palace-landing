Privacy Policy

Memory Palace · Cập nhật 2026-05-02 · Phiên bản 1

Tóm tắt 30 giây: Memory Palace là local-first. Tài liệu thô + wiki output đều ở trên máy bạn. Khi compile/chat, prompt được gửi đến API LLM bạn cung cấp (OpenAI, Anthropic, DeepSeek...). Chúng tôi không có server backend, không lưu data của bạn.

1. Data ở đâu?

1.1 Trên máy bạn (KHÔNG rời máy)

  • Vault — thư mục bạn chọn (ví dụ D:\my-vault). Chứa raw files, wiki Markdown, outputs, logs, config.
  • API keys — lưu trong %APPDATA%\MemoryPalace\.env, mã hoá bằng Windows DPAPI (chỉ tài khoản Windows hiện tại đọc được).
  • License key — %APPDATA%\MemoryPalace\license.json.
  • Install metadata — %APPDATA%\MemoryPalace\install.json (timestamp lần cài đầu, machine fingerprint cho HMAC anti-tamper).

1.2 Rời máy (chúng tôi không thấy)

Khi bạn compile file hoặc chat với wiki, prompt + content được gửi đến provider LLM bạn config qua HTTPS:

  • OpenAI (api.openai.com) — nếu dùng GPT-4o
  • Anthropic (api.anthropic.com) — nếu dùng Claude
  • DeepSeek, Google Gemini, Mistral, Groq... — tuỳ provider bạn chọn

Mỗi provider có chính sách privacy riêng — chúng tôi không can thiệp. Bạn chọn provider nào, data đi tới provider đó. Memory Palace là middleman client-side, không relay qua server của chúng tôi.

1.3 Nếu bạn bật Trial Credits proxy (optional, tương lai)

Khi bạn cấu hình LLM_PROXY_URL để dùng gateway của chúng tôi (giúp bạn dùng app mà không cần mua API key riêng):

  • Prompt + content sẽ đi qua proxy server của chúng tôi → LLM provider thật
  • Chúng tôi không lưu prompt, chỉ log: timestamp + token count + email user (cho usage metering)
  • Log retention: 30 ngày, sau đó tự xoá
  • Tính năng này opt-in, default OFF

3. Tài liệu của bạn KHÔNG bao giờ rời máy bạn

Đây là guarantee cốt lõi của sản phẩm:

  • Vault directory hoàn toàn dưới quyền filesystem của bạn — sync iCloud/Dropbox/Drive nếu bạn chọn, không phải chúng tôi.
  • Memory Palace không có server backend. App là native Tauri + Python sidecar chạy trên máy bạn.
  • Connection duy nhất: tới API LLM bạn cấu hình.
  • Khi bạn delete vault folder, mọi data của bạn biến mất vĩnh viễn — chúng tôi không có copy.

4. Cookie & analytics trên landing page (memorypalace.app)

Landing page dùng:

  • Plausible Analytics (sẽ deploy) — privacy-first analytics, không cookie, không personal data, GDPR/CCPA compliant out-of-the-box. Chỉ track: page views, referrer, country (IP-derived).
  • Cloudflare (CDN/proxy) — lưu IP truy cập trong 24h cho mục đích anti-DDoS.
  • KHÔNG có Google Analytics, Facebook Pixel, hay tracking cross-site nào.

5. Quyền của bạn

  • Yêu cầu xoá data: email hello@memorypalace.app với order ID — chúng tôi xoá record trong D1 trong 7 ngày.
  • Yêu cầu export data: chúng tôi gửi JSON dump tất cả record liên quan email bạn trong 7 ngày.
  • Refund: 14 ngày unconditional. Không cần lý do. Xem Refund policy.
  • License revoke: bạn có thể yêu cầu deactivate license bất cứ lúc nào.

6. Thay đổi policy này
