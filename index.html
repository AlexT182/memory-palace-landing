FAQ

Câu hỏi thường gặp

Memory Palace chạy trên hệ điều hành nào?

Hiện tại Windows 10/11 (64-bit). macOS và Linux trong roadmap Q3 2026. RAM tối thiểu 4 GB, ổ cứng 1 GB cho app + size vault.

Khi cài, Windows hiện cảnh báo "Unrecognized app" — có sao không?

Không sao. Đây là hành vi mặc định với phần mềm chưa code-signed. Memory Palace sẽ có chứng chỉ chính thức ở v1.1 (sau 50 user đầu). Cách bypass: Click "More info" → "Run anyway". Verify integrity: So sánh SHA256 với giá trị publish ở GitHub Release.

Tôi cần API key OpenAI/Anthropic — phải mua riêng à?

Có, đây là mô hình BYOK (Bring Your Own Key) — bạn tự đăng ký và lấy API key, paste vào app. Chi phí thực tế: compile 1 PDF 20 trang ≈ $0.01-0.05 với DeepSeek, $0.05-0.20 với Claude. Sắp ra mắt — Trial Credits: 14 ngày trial KHÔNG cần API key (chúng tôi cấp credit miễn phí qua proxy).

Dữ liệu của tôi có upload lên cloud không?

Không. Memory Palace là 100% local-first. Vault và wiki nằm ở thư mục bạn chọn. Khi compile, prompt được gửi đến API LLM bạn cung cấp (OpenAI/Anthropic/DeepSeek) — bạn kiểm soát hoàn toàn provider này. Chúng tôi không có server backend nào.

"Lifetime" nghĩa là update miễn phí mãi mãi?

Lifetime Personal = mọi bản v1.x miễn phí. Khi ra v2.0 (~18-24 tháng nữa), bạn có thể chọn upgrade với giá ưu đãi (~$20) hoặc giữ v1.x dùng mãi mãi. Mô hình theo Sublime Text, Beyond Compare — không phải subscription trá hình.

Tôi có thể refund không?

Có — 14 ngày unconditional refund. Email hello@memorypalace.app với order ID, tiền về trong 3-5 ngày. Không hỏi lý do, không argue.

Có nhiều vault không? (cho client khác nhau)

Có. Multi-vault native. Mỗi vault là 1 thư mục độc lập với raw/wiki/outputs/settings riêng. Phù hợp consultant 1 vault/client (NDA isolation), researcher 1 vault/topic, founder 1 vault personal + 1 vault competitive.

Memory Palace khác gì Obsidian + plugin AI?

Obsidian + plugin = AI hỏi-đáp trong Obsidian. Memory Palace = dây chuyền sản xuất — drop file vào, AI tự đọc, tự tạo wiki structured (concept atomic, summary, MOC, cross-link), rồi mới chat. Output là Markdown chuẩn Obsidian → bạn có thể mở wiki của Memory Palace bằng Obsidian. Hai tool bổ trợ nhau.
